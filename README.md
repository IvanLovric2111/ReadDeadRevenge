# Red Dead Revenge

## Overview

**Red Dead Revenge** is a dynamic 2D top-down shooter set in the gritty atmosphere of the **Wild West**, where players take on the role of a lone gunslinger fighting for survival. The primary objective is to eliminate **100 enemies**, including skilled **cowboys** and **Eagle Eye** warriors, before facing the **Angry Fatman**, a final boss wielding a devastating machine gun. The game features a realistic weaponry system with detailed reload logic and unique firing characteristics. Players must manage a dynamic inventory, looting crates or fallen enemies for new weapons or beer to restore health. The experience is designed to challenge the player's resource management and tactical movement across an unforgiving frontier.

## Tech Stack

* **Engine:** GameMaker Studio 2
* **Language:** GML (GameMaker Language)
* **Art Style:** 2D Pixel Art

## Getting Started

### 1. For Players

1. Go to the **Releases** section on the right side of this GitHub page.
2. Download the latest `RedDeadRevenge.zip` file.
3. Extract the folder and run `RedDeadRevenge.exe`.
4. **Controls:**
    * `WASD` - Movement
    * `LMB` - Shoot
    * `R` - Reload
    * `Mouse Wheel` - Switch Weapons
    * `P` - Pause
    * `Esc` - Exit

### 2. For Developers (Prerequisites: [GameMaker](https://gamemaker.io/en))

1.  Clone the repository to your local machine and open the `RedDeadRevenge.yyp` file in **GameMaker**.
2.  Edit code as you wish.
3.  Press F5 (or the Play button) to compile and run the game.
4.  **Controls:**
    * `WASD` - Movement
    * `LMB` - Shoot
    * `R` - Reload
    * `Mouse Wheel` - Switch Weapons
    * `P` - Pause
    * `Esc` - Exit
